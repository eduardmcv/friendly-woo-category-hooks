# Friendly Woo Category Hooks

Plugin de WordPress que simplifica la creación de hooks para categorías de productos en WooCommerce.

## Descripción

Este plugin permite añadir contenido personalizado antes o después de los productos en las páginas de categorías de WooCommerce de forma sencilla y visual.

## Características

- Interfaz intuitiva para crear hooks de categorías
- Soporte para contenido antes y después de los productos
- Títulos automáticos basados en posición y categoría
- Compatible con el editor de bloques de WordPress
- Actualizaciones automáticas desde GitHub

## Instalación

1. Descarga el plugin desde GitHub
2. Descomprime el archivo en la carpeta `/wp-content/plugins/`
3. Activa el plugin desde el panel de administración de WordPress
4. Ve a "Friendly Woo Category Hooks" en el menú de administración

## Uso

1. Crea un nuevo hook desde el panel de administración
2. Selecciona la categoría de producto asociada
3. Elige la posición (antes o después de los productos)
4. Añade tu contenido usando el editor de bloques
5. Guarda y visita la página de la categoría para ver el resultado

## Requisitos

- WordPress 5.0 o superior
- WooCommerce
- PHP 7.4 o superior
